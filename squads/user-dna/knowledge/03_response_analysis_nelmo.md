# SISTEMA DE ANÁLISE DE RESPOSTAS
## Estrutura em 4 Camadas

A cada resposta fornecida pelo usuário, analise as seguintes dimensões:

---

### CAMADA 1: CONTEÚDO MANIFESTO
- **O que está sendo dito explicitamente?**
- Identifique: fatos, opiniões, declarações autoafirmativas, histórias compartilhadas
- **Marcadores:** substantivos concretos, tempo verbal no passado, exemplos diretos

---

### CAMADA 2: PADRÕES LINGUÍSTICOS
- **Como a pessoa constrói suas respostas?**
- Identifique vocabulário emocional, estruturas narrativas, metáforas recorrentes
- **Marcadores:** uso de metáforas visuais, repetições, oscilações entre primeira e terceira pessoa

---

### CAMADA 3: CONTEÚDO LATENTE
- **O que está implícito ou ausente?**
- Busque contradições, omissões estratégicas, temas recorrentes não reconhecidos
- **Marcadores:** evasão, desconexão entre emoção e conteúdo, silêncios narrativos

---

### CAMADA 4: INDICADORES EMOCIONAIS
- **Quais emoções estão sendo ativadas?**
- Identifique intensidade, variação, congruência entre fala e emoção
- **Marcadores:** pausas, hesitações, reforço emocional, mudança de tom, afirmações suavizadas ("acho que", "talvez")

---

## PARÂMETROS DE CATEGORIZAÇÃO

Para cada resposta, gere uma codificação nos parâmetros abaixo:

- **Clareza**: Alta / Média / Baixa
- **Profundidade**: Superficial / Funcional / Reflexiva / Reveladora
- **Emoção Dominante**: Alegria / Raiva / Medo / Culpa / Orgulho / Ambivalência / etc.
- **Domínio Principal Ativado**: Um dos 9 domínios (Identidade, Valores...)
- **Insight Detectado**: Sim / Não
- **Contradição Latente**: Sim / Não
- **Resistência Detectada**: Sim / Não

---

## MAPA DE ESTRATÉGIAS DERIVADAS

### CASO 1: ALTA CLAREZA + BAIXA PROFUNDIDADE
- Provoque aprofundamento com pergunta de sondagem ou paradoxo

### CASO 2: CONTRADIÇÃO DETECTADA
- Armazene para futura exploração indireta
- Programe pergunta paradoxal no ciclo seguinte

### CASO 3: INSIGHT DETECTADO
- Estimule elaboração
- Reduza ritmo e reforce percepção do usuário

### CASO 4: RESISTÊNCIA DETECTADA
- Mude o domínio temporariamente
- Volte mais tarde com abordagem lateral

### CASO 5: EMOÇÃO INTENSA
- Espelhe com empatia
- Use perguntas de aprofundamento emocional

---

## APLICAÇÃO
Este sistema de análise alimenta a lógica de seleção de próximas perguntas, além de gerar os dados necessários para compor as mini-sínteses e o perfil final do expert.
