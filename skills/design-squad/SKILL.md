# Design Squad

Squad de design profissional que transforma produto + oferta em experiência visual completa.

## Quando Usar
- "Cria design system do zero"
- "Monta UX flow do meu funil"
- "Gera carrossel pro Instagram"
- "Especifica componente X"
- "Auditoria de design"
- "Gera handoff pra desenvolvedor"

## Comandos
- `/design-squad:create-design-system` - design system completo (cores, typography, components)
- `/design-squad:design-ux-flow` - UX research + fluxo de telas
- `/design-squad:create-carousel` - carrossel Instagram com design system
- `/design-squad:create-component-spec` - especificação de componente
- `/design-squad:generate-handoff` - documentação pra dev
- `/design-squad:audit-design` - auditoria visual + maturidade
- `/design-squad:setup-design-ops` - setup de DesignOps
- `/design-squad:diagnose` - diagnóstico do problema de design
- `/design-squad:review` - review final
